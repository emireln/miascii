## `> whoami`

## `> features`

### в—ј text вҶ’ ascii
- **300+ FIGlet fonts** auto-loaded from the installed catalog (same as TAAG)
- Live preview as you type, horizontal layout modes, max-width wrapping
- Borders (single В· double В· heavy В· dashed В· blocks) В· case folding В· trim
- Random font shuffle В· export `.txt` / `.png`

### в—ј image вҶ’ ascii
- Drop any `png` В· `jpg` В· `webp` В· `gif`
- **Resolution** up to 300 columns В· **character aspect** tuning for your font
- **Density ramps** (standard, blocks, bw, discord, `@в–‘в–“в–’в–‘`, custom)
- **Dithering** (floydвҖ“steinberg В· ordered) В· **Sobel edge detection**
- **Color modes**: monochrome В· full color В· **duotone** (pick shadow + highlight)
- Contrast В· brightness В· gamma В· blur В· flip H/V
- Export `.txt` В· `.png`

### в—ј video & webcam вҶ’ ascii
- Drop any `mp4` В· `webm` В· `mov`, **or** use your **webcam** live
- Real-time rendering at up to **60 fps** (target FPS slider with actual-fps readout)
- All the image-mode controls, plus mirror / flip-v
- **Record** the output to `.mp4` or `.webm` via `MediaRecorder` + canvas capture stream
- Copy any paused frame as text / png

### в—ј 12 languages + RTL
English В· EspaГұol В· PortuguГӘs (BR) В· FranГ§ais В· Deutsch В· Italiano В· Nederlands В· Р СғСҒСҒРәРёР№ В· ж—Ҙжң¬иӘһ В· н•ңкөӯм–ҙ В· дёӯж–Ү В· Ш§Щ„Ш№ШұШЁЩҠШ© (RTL)

### в—ј desktop app
Native-feeling installers for **Windows В· macOS В· Linux** via Electron + electron-builder
- Custom branded installer graphics + multi-resolution icons (16 вҶ’ 1024)
- **System tray** with context menu
- **Close-to-tray** behaviour (toggleable)
- **Settings panel** with: always-on-top, launch at startup, CRT intensity, UI scale, reduce-motion, export/import/reset
- Single-instance lock В· no auto-updater snooping В· fully offline

## `> install`

### Web
```bash
git clone https://github.com/emireln/miascii.git
cd miascii
npm install
npm run dev            # http://localhost:5173
```

### Desktop (dev)
```bash
npm run electron:dev   # opens a native window, hot-reload
```

### Desktop (build your own installer)
```bash
npm run electron:build:win     # вҶ’ release/miascii-x.y.z-setup-x64.exe
npm run electron:build:linux   # вҶ’ release/*.AppImage + *.deb
npm run electron:build:mac     # macOS only, requires Xcode tools
```

> **Want a ready binary?** Grab one from the [**Releases**](https://github.com/emireln/miascii/releases) page.

## `> privacy`

> miascii never uploads anything.

- Every operation (including video processing and webcam frames) runs **locally** via the browser's canvas APIs
- No analytics В· no telemetry В· no accounts В· no remote fonts
- The Electron build opens no network connections except for links you explicitly click
- Your `localStorage` (`miascii:*` keys) holds your preferences; export/import/reset them from Settings
